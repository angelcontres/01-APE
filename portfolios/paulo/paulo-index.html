Sobre mí

¡Hola! Soy estudiante de Ingeniería de Software (cursando el sexto semestre) y un apasionado por la creación de software de calidad.

Para mí, el desarrollo va mucho más allá de escribir líneas de código; implica pensar críticamente en la arquitectura, aplicar principios de Clean Code y Arquitectura Hexagonal, y colaborar en la gestión de equipos para construir soluciones robustas y escalables a largo plazo.
